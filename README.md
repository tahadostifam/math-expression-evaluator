### Run

```bash
crystal run src/main.cr
```

### Build

```bash
crystal build src/main.cr
./main
```

### Examples

-   ```crystal
      (10 + 20) / 3 # -> 10
      10 + 20 / 3 # -> 10
    ```
-   ```crystal
      3 * (9 / 3) # -> 9
      10 * 9 / 3 # -> 30
    ```
-   ```crystal
      10 + 20 + 30 + 40 / 4
    ```
